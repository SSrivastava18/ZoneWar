# 🏃‍♂️ ZoneWar

A gamified mobile run-tracking app designed to make running fun by allowing users to claim regions, compete on leaderboards, and track performance.

---

## 🚀 Features

* 🗺️ **Region Claiming System**
  Users can claim a region by completing a loop (route).

* ⚡ **Competitive Gameplay**
  Other users can reclaim the same region by completing the loop in less time.

* 🏆 **Leaderboard**
  Track top performers and compare rankings.

* 📊 **Performance Analytics**
  View personal stats and compare with other users.

* 📱 **Responsive UI**
  Optimized for smooth performance across Android devices.

---

## 🛠️ Tech Stack

* **Mobile**: React Native, Expo
* **Backend / Services**: Firebase
* **Language**: JavaScript

## 📦 Installation & Setup

### 1. Clone the repository

```bash id="f2r6g1"
git clone https://github.com/SSrivastava18/ZoneWar.git
cd ZoneWar
```

### 2. Install dependencies

```bash id="8jw1gq"
npm install
```

### 3. Start the app

```bash id="r2zq5n"
npx expo start
```

### 4. Run on device

* Scan QR using Expo Go app (Android)
* Or run on emulator
